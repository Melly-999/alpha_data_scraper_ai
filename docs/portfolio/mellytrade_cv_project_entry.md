## 2. Longer CV project entry (English)

**MellyTrade — Safety-first read-only AI trading terminal & portfolio risk
demo (portfolio project).**
Designed and built a full-stack fintech demo intended to showcase safe
trading-system architecture rather than live trading. Implemented a
FastAPI/Pydantic backend with GET-only trading surfaces and a machine-readable
safety endpoint; a React/TypeScript/Vite frontend with a read-only safety
rail, plus a mobile route and a Tauri desktop shell. Added a read-only
Neon/Postgres memory layer with a SQL guard, read-only transactions, and
graceful degradation. Deployed on Render (API) and Vercel (web); maintained CI
(lint, type-check, tests, e2e, security scans) and captured public GET-only
smoke evidence. Delivered through small, reviewed pull requests using an
AI-assisted workflow with documented review gates.
