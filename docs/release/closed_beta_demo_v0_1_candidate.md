# MellyTrade Closed Beta Demo v0.1 Candidate

## Status

Status: Candidate PASS

## Summary

MellyTrade Closed Beta Demo v0.1 is a local read-only, dry-run AI trading research terminal candidate.

It includes:

- terminal overview
- AI scanner/watchlist visual polish
- risk manager read-only posture
- audit/event rail
- broker safe-disconnected/degraded status
- demo/seeded fallback data
- degraded services banner
- browser UI smoke checklist
- closed beta disclaimer and limitations docs
- README screenshot pack

## Safety posture

```text
autotrade=false
dry_run=true
read_only=true
live_orders_blocked=true
max risk <=1%
```

## What is ready

- local backend/frontend startup
- local read-only smoke script
- safety badges and degraded banner
- scanner/watchlist demo UI
- risk manager read-only UI
- audit/event visibility
- broker safe-disconnected view
- README screenshot pack
- closed beta disclaimer/limitations docs
- bug report template

## What is not included

- no live trading
- no broker execution
- no order placement
- no order cancellation
- no order modification
- no production auth
- no production billing
- no investment advice
- no guaranteed profits
- no production deployment SLA
