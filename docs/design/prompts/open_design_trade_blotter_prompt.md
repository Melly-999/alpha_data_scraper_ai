# Open Design Prompt - Trade Blotter Tab

Create a MellyTrade institutional terminal concept for the **Trade Blotter** tab.

Design goal:

- read-only, display-only, advisory-only layout
- compact audit surface for orders and executions
- institutional dark terminal aesthetic
- consistent with MellyTrade brand direction
- no live trading cues
- no order placement controls
- no Buy / Sell / Execute / Place Order buttons
- no broker execution affordances
- no POST / PUT / PATCH / DELETE trading patterns

Include:

- order / execution list
- filter row such as All / Filled / Pending
- compact readable table
- source visibility
- status visibility
- direction visibility
- a premium but operational feel

Generate **3 concept directions** for the same tab:

- Version A: Dense Institutional / operator-heavy
- Version B: Cleaner Executive / premium dashboard
- Version C: Mobile-aware / touch-friendly / iPad-friendly

For each version, show:

- layout structure
- filter treatment
- table density
- state chips / badges
- source and status hierarchy
- mobile or tablet adaptation

Keep the designs serious, crisp, and audit-first.
Do not introduce any execution controls or trading actions.
