# Open Design Prompt - MellyTrade 3 Tab Suite

Create a MellyTrade institutional terminal concept suite for these three tabs:

1. Positions
2. Trade Blotter
3. Backtest Lab

Design goals:

- read-only, display-only, advisory-only layouts
- institutional dark terminal aesthetic
- consistent MellyTrade brand language
- premium, serious, and audit-first
- no live trading cues
- no order placement controls
- no Buy / Sell / Execute / Place Order buttons
- no broker execution affordances
- no POST / PUT / PATCH / DELETE trading patterns

For each tab, generate **3 concept directions**:

- Version A: Dense Institutional / operator-heavy
- Version B: Cleaner Executive / premium dashboard
- Version C: Mobile-aware / touch-friendly / iPad-friendly

Tab requirements:

- Positions: open positions table, closed today table, KPI strip for float PnL / realized PnL / open count, strong status visibility
- Trade Blotter: execution list, All / Filled / Pending filters, compact readable table, source / status / direction visibility
- Backtest Lab: strategy simulation section, configuration card, saved runs, KPI cards, equity curve panel, drawdown panel

Also include:

- a subtle Melly PET assistant concept
- Melly PET motion ideas such as idle float, pulse, blink, hover, gentle wave
- suggested placements for Melly PET such as lower-right assistant bubble, onboarding helper, empty-state helper, loading companion
- a premium desktop app / EXE icon concept that combines Melly PET with the MellyTrade brand in a clean fintech style

Keep the output cohesive across all three tabs.
Do not introduce any execution controls or trading actions.
