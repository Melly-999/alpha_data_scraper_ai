# Frontend Terminal Panel

Use this skill when adding read-only frontend panels to Terminal or AI Workspace.

## Rules

- No order buttons.
- No execute buttons.
- No connect-live buttons.
- No autotrade toggles.
- No dry_run toggles.
- No credential or account ID display.
- Display safety badges prominently.
- Use existing GET-only APIs.
- Fallback or degraded UI is allowed.

## Validation

- Run `npm run build` if frontend files changed.

## UI Guidance

- Keep the panel read-only and advisory.
- Prefer preview cards, status badges, and audit notes.
- Surface unsafe conditions as warnings, not actions.
