# Grok Alpha AI - Investor One-Pager

## EN (Investor)

### What It Is

Grok Alpha AI is a desktop trading copilot for MetaTrader 5 users. It combines live market snapshots, configurable signal logic, and guarded auto-execution in one operator-friendly app.

### Problem Rynkowy

Retail algo traders face a hard trade-off: manual workflows are slow and fragmented, while many auto-bots are opaque and risky.

### Solution

Grok Alpha AI delivers a safe middle path:

- Live visibility: signal, confidence, and runtime state in real time
- Controlled automation: confidence gates, cooldowns, duplicate-signal controls
- Stepwise adoption: paper-safe defaults before live execution

### Why It Wins

- Safe-by-default product posture
- Transparent decision and execution status
- Familiar MT5 environment for fast onboarding

### Target User

- Retail quantitative traders already using MT5
- Semi-automated discretionary traders moving toward automation

### Current Product Proof

- Live GUI controls (start, stop, refresh, interval)
- Continuous mode and config profiles
- MT5 execution module with dry-run and risk limits

### Business Wedge

Start with safety-focused MT5 power users, then expand into broader retail automation via profile templates, execution analytics, and explainability.

### Key Metrics

- Time-to-first-run < 5 minutes
- Paper-safe profile adoption rate
- Error-free session rate
- Conversion from paper to conservative live profile

### Near-Term Roadmap

1. Snapshot and order-intent journaling
2. Signal explainability panel
3. In-GUI profile switching
4. MT5 readiness diagnostics (login/symbol/account)

### Risks

- MT5 availability and local terminal state
- User misconfiguration in live mode

Mitigation: dry-run-first defaults, strict runtime guardrails, explicit execution statuses.

## PL (Wersja Skrócona)

### Czym Jest Produkt

Grok Alpha AI to desktopowy copilot tradingowy dla użytkowników MetaTrader 5. Łączy podgląd rynku na żywo, konfigurowalną logikę sygnałów i kontrolowaną auto-egzekucję w jednej aplikacji.

### Problem

Traderzy detaliczni mają zwykle wybór między wolnym, ręcznym workflow a nieprzejrzystymi botami o podwyższonym ryzyku.

### Rozwiązanie

Grok Alpha AI oferuje bezpieczny środek:

- Pełna widoczność na żywo: sygnał, confidence, stan runtime
- Kontrolowana automatyzacja: progi confidence, cooldown, blokada duplikatów sygnału
- Stopniowe wdrożenie: bezpieczny tryb paper jako domyślny

### Przewaga

- Bezpieczeństwo jako domyślne ustawienie
- Przejrzysty status decyzji i egzekucji
- Integracja z MT5, więc niski próg wejścia dla grupy docelowej

### Klient Docelowy

- Zaawansowani traderzy detaliczni korzystający z MT5
- Traderzy półautomatyczni przechodzący w kierunku automatyzacji

### Dowód Produktowy (Dziś)

- Live GUI (start, stop, refresh, interwał)
- Tryb ciągły i profile konfiguracyjne
- Moduł MT5 z dry-run i limitami ryzyka

### Klin Wejścia Na Rynek

Najpierw użytkownicy MT5 nastawieni na bezpieczeństwo, następnie rozszerzenie oferty przez profile strategii, analitykę egzekucji i warstwę explainability.

### Kluczowe Metryki

- Czas do pierwszego uruchomienia < 5 minut
- Odsetek użytkowników startujących od profilu paper-safe
- Odsetek sesji bez błędów runtime
- Konwersja z paper na profil real-conservative

### Najbliższa Mapa Drogowa

1. Dziennik snapshotów i intencji zleceń
2. Panel wyjaśniania sygnału
3. Przełączanie profili z poziomu GUI
4. Diagnostyka gotowości MT5 (login/symbol/konto)

### Ryzyka

- Zależność od dostępności środowiska MT5
- Błędna konfiguracja użytkownika w trybie live

Mitigacja: domyślny dry-run, twarde guardraile runtime, jawne statusy egzekucji.
