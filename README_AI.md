# AI Trading System (Alpha Data Scraper AI)

## 🚀 Overview
This project is evolving into a full AI-powered trading system using:
- Python (FastAPI)
- MetaTrader 5
- Ollama (local LLMs)
- Claude (AI orchestration)

## 🧠 Features (Planned)
- Data scraping (market + external signals)
- AI signal generation (BUY/SELL/HOLD)
- Multi-agent system (risk, execution, analysis)
- Automated trading via MT5
- Dashboard (future)

## ⚙️ Tech Stack
- Python
- FastAPI
- Pandas / NumPy
- MetaTrader5
- Ollama (Llama3 / Phi3)
- Claude
